Work

Ask Prags

A cool Information Retrieval Based Question Answering system built using Transformers.
Technologies and concepts used: Python and Ruby on Rails.
Main Modules used: Transformers, Tensorflow, Torch, and Wikipedia.
The project is hosted in this link


Virtual Pricer

An option pricing model using concepts of Reinforcement Learning.
Technologies and concepts used: Python and Ruby on Rails.
Main Modules used: Tensorflow, Numpy and QuantLib.
The project is hosted in this link


LSTM

Well, it's not LSTM built from scratch. It's an effort to predict the movement and prices of stock using data such as Financial Statements like Income Statement, Balance Sheets and Cash Flow Statements.
Technologies and concepts used: Python.
Main Modules used: Tensorflow, Gensim, NLTK and Numpy
The project is hosted in this link


TCEM

This is a model which is built to use for Text Classification using Expectation Maximization
Technologies and concepts used: Python and Expectation Maximization(EM).
Main Modules used: Numpy, Pandas, Matplot Library, Scikit Learn and NLTK
The project is hosted in this link


HMMer

implemented Multiple Sequence Alignment using HMM(Hidden Markov Model). The model was used to analyze if the given sequence belongs to a particular family of genes. Baum-Welch algorithm was used to train the HMM and Viterbi algorithm was used to predict with a trained HMM.
Technologies and concepts used: Python and Hidden Markov Model(HMM)
Main Modules used: Numpy
The project is hosted in this link
